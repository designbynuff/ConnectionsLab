Pulse Clock

An LED wall clock that lets you become time.

Made in Interface Lab and Exploratory Making at IMA Low Res, Summer 2023.

Clock Mode

Pulls live time data from the cloud and displays the time in hours, minutes and seconds.

Pulse Mode

Uses a heart rate sensor to create an animation from your heart rate.

Components

  • Pulse Sensor Amped
  • DIYMall 60 pixel LED ring
  • Arduino Nano 33IOT
  • 3D Printed Clock Chassis
  • Laser Cut Acrylic Clock Face
